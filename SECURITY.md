# Security Policy

## Reporting A Vulnerability

Please report security vulnerabilities privately by emailing
security@openbase.cloud.

Include as much detail as you can reasonably provide:

- Affected component or command
- Steps to reproduce
- Expected and actual behavior
- Impact assessment
- Relevant logs, screenshots, or proof-of-concept details

Please do not publicly disclose the issue until we have had a reasonable
opportunity to investigate and coordinate a fix.

## Scope

This policy covers vulnerabilities in the Openbase Coder CLI and its local
runtime services in this repository.

For issues in third-party dependencies, report to the dependency maintainer as
well as to Openbase if the issue affects Openbase Coder users.
